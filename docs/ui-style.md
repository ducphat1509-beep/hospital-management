UI Style Guide (Swing):

Theme:
- Light modern dashboard
- Soft colors (white, light blue, gray)
- Rounded panels

Layout:
- Sidebar on left
- Topbar on top
- Content area uses CardLayout

Components:
- Use JPanel as card container
- Use JTable for data
- Use JButton with consistent size

Spacing:
- Padding: 10-20px
- Use empty borders

Rules:
- No business logic in UI
- UI calls Service only
- Each panel = 1 class
